Accessibility Analyzer Extension

**Note: this extension has been developed in half an hour with a lot of help from AI. It might not be the best tool out there, but it is a good starting point for analyzing your pages for accessibility issues. Feel free to submit a PR with potential optimizations.**

How to use it?
- Download the code
- Open Chrome
- Click the three dots in the upper right corner -> Extensions -> Manage extensions
- Enable Developer mode in the upper right corner
- Click Load unpacked in the upper left corner
- Select the folder with the code that you have just downloaded
- When the extension is uploaded, you should be able to use it by clicking the Extensions button in Chrome, clicking the Accessibility Analyzer extension, and then clicking the Analyze Page button
- The extension will highlight all page elements which are not accessibility-friendly
